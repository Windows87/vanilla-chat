## How to Run
### Clone Repository
```bash
# Clone this repository
git clone https://github.com/Windows87/vanilla-chat/
# Go into repository
cd vanilla-chat
```

### Start Server
```bash
# Install dependencies
npm install
# Start
node index.js (or nodemon)
```
